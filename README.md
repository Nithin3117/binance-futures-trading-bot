# Binance Futures Trading Bot

This project was developed as part of a Python Developer internship assignment.

The application connects to the Binance Futures Testnet and allows users to place Market and Limit orders through the command line.

## Features

- Place Market Orders
- Place Limit Orders
- Supports BUY and SELL
- Command line interface using argparse
- Input validation
- Logging
- Error handling
